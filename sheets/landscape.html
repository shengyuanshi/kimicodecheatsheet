Install

Script install is recommended. npm requires Node.js 24.15.0+.

macOS / Linux Script install curl -fsSL https://code.kimi.com/kimi-code/install.sh | bash
Windows PowerShell No Node.js required irm https://code.kimi.com/kimi-code/install.ps1 | iex
npm @moonshot-ai/kimi-code npm install -g @moonshot-ai/kimi-code
pnpm Alternative package manager pnpm add -g @moonshot-ai/kimi-code

Fast Start

kimi in a repo. Use /init once, /plan for risky work, and /compact when context gets long.

Trust Rule

--yolo is for trusted workspaces only. Keep approvals on when touching unfamiliar code or production data.

CLI Command
